# What are .files? (a.k.a. dotfiles)

Configuration files in a $HOME directory, such as:

> - .bash_profile
> - .gitconfig
> - .tmux.conf
> - .vimrc

---

# What does Ellipsis do?

Ellipsis is a package manager for dotfiles.

---

# Dependencies

Just 3 programs are required:

- bash
- curl
- git
